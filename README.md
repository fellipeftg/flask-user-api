# API de Grenciamento de Usuários

API REST desenvolvida com **Flask** e **PostgreSQL** para gerenciamento de usuários.

## Tecnologias

- Python
- Flask
- SQLAlchemy
- PostgreSQL

## Funcionalidades

- Criar usuário
- Listar usuários
- Buscar usuário por ID
- Atualizar usuário
- Deletar usuário

## Instalação

Clone o projeto:

git clone https://github.com/seu-usuario/flask-user-api.git

Crie ambiente virtual:

python -m venv venv

Instale dependências:

pip install -r requirements.txt

Configure o `.env`

Execute:

python run.py
